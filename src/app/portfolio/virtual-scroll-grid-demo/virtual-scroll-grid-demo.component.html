Virtual Scroll Grid Demo

VirtualScrollGridComponent is useful for displaying a large number of images. Tile elements are not mounted until the user scrolls into the viewport. This allows for lazy loading of images and quick render times.

The width and height of the tiles automatically adjust based on the virtualScrollGridComponent's size. This can be used to load lower resolution images when higher resolution images are not required.

is based on cdk-virtual-scroll-viewport and a ResizeSensor
